# Student Notes Security Assignment

This project is a simple Flask-based student notes application created for the Cyber Security Base course assignment.

The application contains five intentionally introduced security vulnerabilities based on the OWASP Top 10 2021.

The vulnerabilities are intentionally included for educational purposes. Each vulnerability is documented together with a description of the problem and a proposed secure implementation.

---

## Application Features

The application provides the following functionality:

- User registration
- User login
- User logout
- Creating notes
- Viewing notes
- Deleting notes

---

## Technologies

- Python
- Flask
- SQLite
- Jinja2
- HTML
